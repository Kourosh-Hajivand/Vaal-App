# آیکون‌های آب‌وهوا

آیکون‌های پیش‌بینی را اینجا بگذار و در `src/constants/weatherIcons.ts` require کن.

**نام فایل‌ها (اختیاری — هر نامی که در weatherIcons ست کنی):**

| کلید           | وضعیت     |
|----------------|-----------|
| `clear`        | آفتابی   |
| `partly-cloudy`| نیمه‌ابر |
| `cloudy`       | ابری     |
| `fog`          | مه       |
| `drizzle`      | نم نم باران |
| `rain`         | باران    |
| `snow`         | برف      |
| `thunderstorm` | رعد و برق |
| `unknown`      | نامشخص   |

فرمت پیشنهادی: PNG با پس‌زمینه شفاف، اندازه تقریبی ۵۶×۵۶ یا ۲۸×۲۸ پیکسل.
